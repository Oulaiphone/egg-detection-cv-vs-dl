# Images

This folder contains sample images, results, and diagrams from the egg detection project.

## Folder Structure

```
images/
├── README.md           # This file
├── results/            # Detection output screenshots

```

## Contents

### samples/
Sample egg images used for detection testing. Include a variety of:
- Single eggs
- Multiple eggs
- Different lighting conditions
- Various backgrounds (conveyor belt, floor, nest)

### results/
Screenshots showing detection results:
- Traditional CV detection with egg counts
- Faster R-CNN detection with bounding boxes and confidence scores
- TensorBoard training graphs

### architecture/
Diagrams illustrating:
- Traditional CV pipeline workflow
- Faster R-CNN architecture
- Comparison charts
